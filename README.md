Реализация jQuery - ots
===========================================

## Установка:

### Склонировать данный репозиторий
git clone https://github.com/newyorrker/ots.git

### Установить пакеты
в директории репозитория выполнить npm install

### Запустить gulp
gulp sass
или
gulp (default)
